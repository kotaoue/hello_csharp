# Hello World

A simple Hello World program in C#.

## Features

- **C#** — A modern, statically typed, object-oriented language developed by Microsoft
- **Cross-platform** — Runs on Linux, macOS, and Windows via .NET
- **Console app** — Minimal setup; no web server or framework required
- **Standard library** — Uses built-in `Console`, `DateTime`, and `Random` without any external dependencies
- **String interpolation** — Leverages C# `$"..."` syntax for readable, inline expressions

## Build & Run

```sh
cd helloworld
dotnet run

Hello, World!
```
